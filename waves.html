Anote In Waves Terms

  • Home
  • Waves

Anote is a Waves fork that uses some of Waves tech and has its own simple wallet (anote.one) implementation and "mobile mining" solution.

It started with 1000 premined anotes to allow settings up the first node as requested by the Waves protocol and then used similar principles as Neutrino node decentralization program to mine new anotes and lease it back to new nodes. Same principle is also used to create the fund for "mobile mining". Leases are currently done by the project developer, but a DAO will be built on the "community" address and leases will be done by voting in the near future.

Mobile mining works in a way that a percentage of Anote block reward goes to a designated address for mobile mining. Users then trigger mining on their mobile phones by entering the daily code published together with a daily ad on our Telegram channel Anote Today. Ads on Telegram channel are purchased through auctions and paid with anotes using Anote Today tool, thus creating a built in demand. Down the road we will enable desktop mining and create an advertising network for Telegram channels. This will provide us with targeting audience by the interest and language.

We also plan to add additional way of mobile mining - asking users questions for the purpose of fully decentralized oracles. Questions like the price of Bitcoin (or other cryptocurrencies) at particular time on particular exchange, questions like sports matches results etc.

Mobile mining is a centralized point, but we will build features to make it possible to change the address for mobile mining fund by voting and replacing the developer / sysadmin that maintains it.
